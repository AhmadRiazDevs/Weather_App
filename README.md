# Weather App 2.0

A lightweight, client-side Weather App that displays current weather data for a chosen location. Built with plain HTML, CSS, and JavaScript for easy customization and local hosting.

## Features

- Simple, responsive UI
- Live weather display using browser-based requests (API key not included)
- Separate styles for premium and base themes

## Getting Started

1. Clone or download the project files.
2. Open `index.html` in a web browser (double-click or serve from a local static server).

No build tools or package managers are required.

## Files

- `index.html` — Main HTML file and app entry point.
- `script.js` — JavaScript logic for fetching and displaying weather data.
- `style.css` — Main stylesheet for the app UI.
- `premium.css` — Optional premium/alternate styles.
- `1.css` — Additional stylesheet (check usage in `index.html`).

## Usage

- To run locally, simply open `index.html` in a browser.
- If the app uses a weather API, add your API key inside `script.js` or configure it according to any documented instructions in the code.

## Customization

- Modify styles in `style.css` or `premium.css` to change the look.
- Update `script.js` to change API endpoints or data handling.
